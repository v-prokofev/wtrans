# Протокол обмена с датчиком ДВУ-01 (РЭ МРАШ.416136.03.001)

## А.1. Режим командной строки
Вход в режим командной строки осуществляется командой `OPEN`. 
Выход — командой `CLOSE` или автоматически через 90 секунд бездействия.

Приглашение системы:
```text
> # поле командной строки
```

## А.2. Команды и форматы
- **Регистр**: Не имеет значения.
- **Параметры**: Отделяются пробелом.
- **Строковые параметры**: Обрамляются двойными кавычками `"`.
- **Завершение команды**: Символ `<CR>` (код `0x0D`).
- **Ответы**: Заканчиваются `<CR><LF>` (`0x0D 0x0A`).
- **Статус успеха**: `OK` или информационное сообщение.
- **Статус ошибки**: `ERROR` или `UNKNOWN COMMAND`.

## А.3. Автономные команды
Выполняются вне сессии (когда командная строка закрыта).
- `OPEN` — команда открытия сессии.

## А.4. Сетевые команды
Формат: `@id команда [параметры]`
- `id` — сетевой адрес (0..254).
- `id = 0` — широковещательный адрес (ответ не формируется).

## А.5. Системные команды

### А.5.1. OPEN <id> (Вход)
Открывает сессию с устройством.
- **Пример**: `OPEN 1`
- **Ответ**: `1, 1.0.0, DVU SESSION OPENED >`

### А.5.2. CLOSE (Выход)
Закрывает текущую сессию.

### А.5.3. HELP
Список доступных команд.

### А.5.4. RESET
Программная перезагрузка устройства.

### А.5.5. BOOT
Переход в режим загрузчика (обновление ВПО).

### А.5.6. DATE [YYYY-MM-DD[Thh:mm:ss]]
Установка или чтение даты/времени.

## А.6. Команды идентификации
- `ID [id]` — Установка сетевого адреса.
- `NAME [system name]` — Установка имени устройства (до 22 символов).
- `VER` — Версия прошивки.
- `SNUM` — Серийный номер.

## А.7. Команды запроса данных

### А.7.1. AMES [msgid [interval]]
Установка режима автоматической отправки сообщений.
- `interval` — 5..3600 секунд. `0` — выключено.

### А.7.2. MES [msgid] (Псевдоним: M)
Запрос информационного сообщения.
- **Пример запроса**: `M`
- **Пример ответа**: `0 6.15 5.94 9.53 3.41 303 298 358 0`
- **Формат сетевой команды**: `@1 M`
- **Ответ сетевой команды**: `@2:0 0 6.15 5.94 9.53 3.41 303 298 358 0`

> **Примечание**: Значение `-99.99` или `-9999` означает отсутствие данных или ошибку.

## А.8. Сервисные настройки
- `BAUD [baudrate]` — Скорость (9600, 19200, 38400, 57600, 115200). Вступает в силу после RESET.
- `CONF` — Вывод всех настроек.
- `INIT {CONF|DEF|FCAL}` — Сброс к заводским настройкам.
- `STAT` — Состояние системы.
- `INTV [interval]` — Интервал усреднения ветра (1..60 мин).
- `GINTV [interval]` — Интервал усреднения порывов (1..5 сек).
- `HEAT [{OFF|ON|AUTO|PWRSAVE}]` — Управление обогревом.
- `NCORR [offset]` — Поправка направления (-180..180 град).
